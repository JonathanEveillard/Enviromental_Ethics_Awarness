# Gallery Images

Add your gallery images here with the following naming convention:
- `pic1.jpg`
- `pic2.jpg`
- `pic3.jpg`
- `pic4.jpg`
- `pic5.jpg`
- `pic6.jpg`
- ... and so on (pic7.jpg, pic8.jpg, etc.)

## Supported Formats
- JPG/JPEG
- PNG
- WebP
- GIF

## Image Requirements
- Recommended size: 500x500px or larger for best quality
- Any aspect ratio works (will be displayed as square in grid)
- File size: Keep under 2MB for optimal performance

## How It Works
The gallery automatically displays all images in the `pic1.jpg`, `pic2.jpg`, etc. pattern. Simply add your images to this folder, and they'll appear in the gallery on the Research page.

### Gallery Features
- **Grid Layout**: Instagram/Pinterest-style responsive grid (2 columns on mobile, 3 on tablet, 4 on desktop)
- **Hover Effect**: Images zoom on hover
- **Lightbox Modal**: Click any image to open a full-size viewer
- **Navigation**: Use Previous/Next buttons or thumbnail strip to browse
- **Image Counter**: See current position (e.g., "1 / 6")

## To Add More Images
1. Add images: `pic1.jpg`, `pic2.jpg`, etc. to this folder
2. They'll automatically appear in the gallery grid
3. The component will dynamically adjust the gallery based on how many images are present

## Notes
- Images are displayed in order of their number
- Maximum recommended gallery size: 20-30 images for performance
- The gallery is fully responsive and works on all devices
